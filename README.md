# Chrome擴充功能-YT直播聊天室直接前往觀眾的頻道

## 動機

我平時喜歡在各個vtuber直播間D台，每當有其他vtuber主播在我正在D的台留言時，我如果想看他的頻道我必須手動去複製他的名字貼到搜尋欄，聊天室有時跑的很快，不是很好去複製名稱，所以我就做了按鈕，點擊就可以直接到觀眾的頻道了。

## 操作例子

游標懸停在聊天室留言上，右手邊就會有前往頻道按鈕，點擊即可。

## 安裝教學

### 1. 下載本專案
* 下載並解壓縮

### 2. 開啟瀏覽器擴充功能頁面
* 在 Chrome 網址列輸入 `chrome://extensions/` 按下Enter。

### 3. 開啟開發人員模式
* 將頁面右上角的 **「開發人員模式」(Developer mode)** 開關切換為 **開啟**。

### 4. 載入擴充功能
* 點擊左上角的 **「載入未封裝項目」(Load unpacked)**。
* 在彈出的視窗中，選擇你剛解壓縮的**資料夾**。
