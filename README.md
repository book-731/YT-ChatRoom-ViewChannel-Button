### Chrome擴充功能-YT直播聊天室直接前往觀眾的頻道

### 操作例子

游標懸停在留言上，右手邊就會有前往頻道按鈕，點擊即可。

#### 動機

我平時喜歡在各個vtuber直播間D台，每當有其他vtuber主播在我正在D的台留言時，我如果想看他的頻道我必須手動去複製他的名字貼到搜尋欄，聊天室有時跑的很快，不是很好去複製名稱，所以我就做了按鈕，點擊就可以直接到觀眾的頻道了。
